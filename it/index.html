Browser per Windows

Disponibile come Setup o Portable, pronto in pochi istanti e pensato per l'uso quotidiano.

Come funziona

Un flusso breve e pulito: installa, avvia, continua a lavorare nella sidebar. Niente testo demo, solo una configurazione reale per l'uso quotidiano.

01

Installa

Scarica il setup o la versione portable, avvia e apri il browser. Nessun passaggio inutile tramite una pagina demo — sei subito operativo.

02

Usa Sidekick

La sidebar resta aperta mentre navighi. Chatta, chiedi, riassumi, agisci.

03

Lavora con Spaces

Separa i progetti in modo pulito, così ricerca, attività e cronologia non si confondono.

Documentazione

Le informazioni più importanti per il release sono già documentate qui. Così nessuno deve saltare tra sito, README e pagina di rilascio.

Download

  • Il download passa direttamente da lastbrowser.com — senza una pagina GitHub in mezzo.
  • Usa Setup se vuoi l'installazione classica con integrazione degli aggiornamenti.
  • Usa Portable se vuoi avviare Lastbrowser senza installazione.

Aggiornamenti

  • L'app recupera i metadati degli aggiornamenti tramite /downloads/latest.yml.
  • I file di release vengono distribuiti direttamente da questo dominio.
  • Così il flusso di aggiornamento resta chiaro, stabile e facile da documentare.

Privacy

  • Sidekick può funzionare in locale o collegarsi alle tue chiavi personali.
  • Rimani tu a controllare il modo in cui lavora l'assistente.
  • Il sito spiega il prodotto, non il materiale di branding.

Aiuto

  • Inizia dalla sidebar e crea uno Space se sei al primo utilizzo.
  • Se stai verificando un download, scegli Setup per l'installazione o Portable per test rapidi.
  • Le pagine più importanti sono collegate qui direttamente.
