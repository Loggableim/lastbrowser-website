Comment l’utiliser

Trois étapes pour le quotidien : installer, utiliser Sidekick, séparer proprement vos projets dans Spaces.

01

Installer

Téléchargez Setup ou Portable, lancez le programme, ouvrez le navigateur. Aucun détour par une page de démonstration — vous êtes productif immédiatement.

02

Utiliser Sidekick

La barre latérale reste ouverte pendant votre navigation. Discutez, demandez, résumez, agissez.

03

Travailler avec Spaces

Vous séparez vos projets proprement pour que la recherche, les tâches et l’historique ne se mélangent pas.

Infos de version

Les informations de version essentielles sont documentées ici, directement — courtes, claires et sans détour par d’autres pages.

Téléchargement

  • Le téléchargement passe directement par lastbrowser.com.
  • Utilisez Setup si vous souhaitez l’installation classique avec intégration des mises à jour.
  • Utilisez Portable si vous voulez lancer Lastbrowser sans installation.

Mises à jour

  • L’application récupère son manifeste de mise à jour via /downloads/latest.yml.
  • Les fichiers de version sont servis directement depuis ce domaine.
  • Le flux de mise à jour reste ainsi clair, stable et facile à documenter.

Confidentialité

  • Sidekick peut fonctionner localement ou être relié à vos propres clés.
  • Vous gardez le contrôle sur le fonctionnement de l’assistant.
  • Le site explique le produit, pas le dossier de marque.

Aide

  • Commencez par la barre latérale et créez un Space si vous débutez.
  • Si vous vérifiez un téléchargement, prenez Setup pour l’installation ou Portable pour des essais rapides.
  • Les pages les plus importantes sont liées ici directement.
