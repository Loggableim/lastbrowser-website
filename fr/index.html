Comment ça marche

Un flux court et propre : installer, lancer, continuer dans la barre latérale. Pas de texte de démonstration, mais une vraie configuration pour le quotidien.

01

Installer

Téléchargez Setup ou Portable, lancez-le, ouvrez le navigateur. Pas de détour par une page de démonstration — vous êtes opérationnel immédiatement.

02

Utiliser Sidekick

La barre latérale reste ouverte pendant votre navigation. Discutez, demandez, résumez, agissez.

03

Travailler avec Spaces

Séparez proprement vos projets pour éviter de mélanger recherche, tâches et historique.

Documentation

Les informations essentielles pour la version sont documentées ici, directement. Ainsi, personne n’a besoin de naviguer entre le site, le README et la page de version.

Téléchargement

  • Le téléchargement passe directement par lastbrowser.com — sans page GitHub intermédiaire.
  • Utilisez Setup si vous voulez l’installation classique avec intégration des mises à jour.
  • Utilisez Portable si vous souhaitez lancer Lastbrowser sans installation.

Mises à jour

  • L’application récupère ses métadonnées de mise à jour via /downloads/latest.yml.
  • Les fichiers de version sont diffusés directement depuis ce domaine.
  • Ainsi, le flux de mise à jour reste clair, stable et facile à documenter.

Confidentialité

  • Sidekick peut fonctionner localement ou être connecté avec vos propres clés.
  • Vous gardez le contrôle sur la manière dont l’assistant travaille.
  • Le site présente le produit, pas le dossier de marque.

Aide

  • Commencez par la barre latérale et créez un Space si vous découvrez l’application.
  • Si vous vérifiez un téléchargement, utilisez Setup pour l’installation ou Portable pour des tests rapides.
  • Les pages les plus importantes sont directement liées ici.
