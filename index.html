Dokumentation

Die wichtigsten Informationen für den Release sind direkt hier dokumentiert. So muss niemand zwischen Website, README und Release-Seite hin- und herspringen.

Download

  • Der Download läuft direkt über lastbrowser.com — ohne GitHub-Seite dazwischen.
  • Nutze Setup, wenn du die klassische Installation mit Update-Integration willst.
  • Nutze Portable, wenn du Lastbrowser ohne Installation starten möchtest.

Updates

  • Die App bezieht ihr Update-Metadatum über /downloads/latest.yml.
  • Die Release-Dateien werden direkt von dieser Domain ausgeliefert.
  • So bleibt der Update-Flow klar, stabil und leicht zu dokumentieren.

Datenschutz

  • Sidekick kann lokal laufen oder mit deinen eigenen Schlüsseln verbunden werden.
  • Du behältst die Kontrolle über die Arbeitsweise des Assistenten.
  • Die Website erklärt das Produkt, nicht die Brand-Mappe.

Hilfe

  • Starte mit der Sidebar und lege einen Space an, wenn du neu einsteigst.
  • Wenn du einen Download prüfst, nimm Setup für die Installation oder Portable für schnelle Tests.
  • Die wichtigsten Seiten sind hier direkt verlinkt.
