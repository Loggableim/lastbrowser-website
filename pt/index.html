Como trabalha com ele

Um fluxo curto e simples: instalar, iniciar e continuar a trabalhar na barra lateral. Não é texto de demonstração — é uma configuração real para o dia a dia.

01

Instalar

Descarregue o setup ou a versão portable, abra o navegador. Sem desvio para uma página de demonstração — entra logo em produção.

02

Usar Sidekick

A barra lateral mantém-se aberta enquanto navega. Conversar, pedir esclarecimentos, resumir, agir.

03

Trabalhar com Spaces

Separe projetos com clareza, para que pesquisa, tarefas e histórico não se misturem.

Documentação

As informações essenciais para este lançamento estão documentadas aqui mesmo. Assim, ninguém precisa andar entre o website, o README e a página de lançamento.

Descarregamento

  • O download é feito diretamente em lastbrowser.com — sem uma página do GitHub pelo meio.
  • Use Setup se quiser a instalação clássica com integração de atualizações.
  • Use Portable se quiser iniciar o Lastbrowser sem instalação.

Atualizações

  • A aplicação obtém o metadado de atualização através de /downloads/latest.yml.
  • Os ficheiros de lançamento são servidos diretamente por este domínio.
  • Assim, o fluxo de atualizações mantém-se claro, estável e fácil de documentar.

Privacidade

  • O Sidekick pode correr localmente ou ser ligado com as suas próprias chaves.
  • O controlo sobre a forma de trabalho do assistente permanece consigo.
  • O website explica o produto, não a narrativa da marca.

Ajuda

  • Comece pela barra lateral e crie um Space se estiver a dar os primeiros passos.
  • Se quiser validar um download, escolha Setup para instalação ou Portable para testes rápidos.
  • As páginas mais importantes estão ligadas aqui mesmo.
