Docs

The key release information is documented right here. That way nobody has to bounce between the website, the README, and the release page.

Download

  • The download is served directly via lastbrowser.com — no GitHub page in between.
  • Use Setup if you want the classic installation with update integration.
  • Use Portable if you want to start Lastbrowser without installing it.

Updates

  • The app reads its update metadata from /downloads/latest.yml.
  • The release files are served directly from this domain.
  • That keeps the update flow clear, stable, and easy to document.

Privacy

  • Sidekick can run locally or connect with your own keys.
  • You stay in control of how the assistant works.
  • The website explains the product, not the brand book.

Help

  • Start with the sidebar and create a Space if you're new to it.
  • If you're checking a download, use Setup for installation or Portable for quick tests.
  • The most important pages are linked directly here.
