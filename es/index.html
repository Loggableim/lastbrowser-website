Lo que incluye

Todo lo que necesitas en un navegador productivo está visible en la página y construido dentro del producto: Chat, Tasks, Skills, Spaces, Memory y el flujo clásico del navegador.

Chat sin saltar entre pestañas

Haz preguntas, entiende contenidos, redacta textos y revisa decisiones — justo al lado de la página.

Tasks siempre a la vista

Crea tareas desde el contexto, priorízalas y déjalas abiertas hasta que estén terminadas.

Skills listos para iniciar

Los flujos integrados para investigación, escritura, análisis y más están listos al instante.

Así se usa

Un flujo corto y limpio: instalar, abrir, seguir trabajando desde la barra lateral. Nada de texto de demo, sino una configuración real para el día a día.

01

Instalar

Descarga Setup o Portable, ábrelo y lanza el navegador. Sin rodeos por una demo: directo a la productividad.

02

Usar Sidekick

La barra lateral permanece abierta mientras navegas. Chatea, pregunta, resume y actúa.

03

Trabajar con Spaces

Separa los proyectos con orden para que la investigación, las tareas y el historial no se mezclen.

Documentación

La información clave del lanzamiento está documentada aquí mismo. Así nadie tiene que saltar entre la web, el README y la página del lanzamiento.

Descarga

  • La descarga se realiza directamente desde lastbrowser.com — sin una página de GitHub intermedia.
  • Usa Setup si quieres la instalación clásica con integración de actualizaciones.
  • Usa Portable si prefieres iniciar Lastbrowser sin instalarlo.

Actualizaciones

  • La app obtiene su metadato de actualización a través de /downloads/latest.yml.
  • Los archivos del lanzamiento se sirven directamente desde este dominio.
  • Así el flujo de actualización se mantiene claro, estable y fácil de documentar.

Privacidad

  • Sidekick puede ejecutarse localmente o conectarse con tus propias claves.
  • Tú mantienes el control sobre el funcionamiento del asistente.
  • La web explica el producto, no el dossier de marca.

Ayuda

  • Empieza por la barra lateral y crea un Space si acabas de llegar.
  • Si vas a comprobar una descarga, usa Setup para la instalación o Portable para pruebas rápidas.
  • Las páginas más importantes están enlazadas aquí mismo.
